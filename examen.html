MANUAL DE ETIQUETAS HTML

Etiqueta Descripcion Ver codigo Ver en browser
bdo El elemento HTML bdo permite a los autores anular el algoritmo bidireccional para fragmentos específicos de texto. Lo que básicamente hace es especificar la dirección del texto en su contenido mediante el atributo "dir".
blockquote El elemento HTML inserta citas en forma de bloque. Como es un elemeneto a nivel de bloque es útil, por ejemplo, para insertar citas largas que se dividen en párrafos. Los agentes de usuario generalmente muestran el texto dentro de este elemento con un márge izquierdo mayor.
canvas Se usa para dibujar gráficos en pantalla
caption El elemento HTML caption provee un título para la tabla en la que es declarado. Usualmente es mostrado encima de la tabla (predeterminadamente) y con un tamaño de fuente mayor.
cite El elemento HTML cite inserta una cita o referencia a otros recursos. Es mayormente utilizado para hacer referencia a otros autores al realizar una cita.
col El elemento HTML col es usado en las tablas para proporcionar atributos comunes a todas las celdas en una columna o grupo de columnas
colgroup El elemento HTML colgroup es usado en tablas para agrupar columnas.
datalist Especifica en un input una lista pre-definida de opciones
dd Define la descripción de un ítem en una lista de definición
wbr Define un posible salto de linea
video la etiqueta VIDEO también tiene un peso muy fuerte en HTML5 por que viene a facilitarnos las cosas en cuanto a la necesidad de agregar contenido audio visual en nuestra pagina Web.
var El elemento HTML var contiene texto que supone ser la instancia de una variable o el argumento de un programa.
track Esta etiqueta especifica las pistas de texto para los elementos multimedia (
thead El elemento HTML thead define una sección de encabezado en una tabla (elemento HTML table). Cuando está presente, debe contener una o más filas que se supone, proveen información de encabezado para las celdas de datos.
tbody Sobreescribe la dirección del texto
sup Define un texto que es superíndice
small Define texto pequeño
param Define un parámetro para un objeto
mark Define texto resaltado o marcado
kbd Define entrada del teclado
